# Level up

## Feature:
When a character reaches a certian amount of collected expreience, the character will level up. This will increase some predifined statistics.

## Scenario
A user named Lil' Frank has played a lot and got a good grip of the game's meta. He concentrates on leveling up his characters by using them more, over the others on this team.

## Given
A character with a level.

## When
Using the character.

## Then
The character's level will be one integer higher than the starting state.

# But
The character cannot be the maximum level
